# pratikreddy.com

Public suite site for `pratikreddy.com`.

The site is built as a Vercel-ready Next.js app. Its data boundary is the metadata-only public feed generated by:

```bash
npm run build-public-feed
```

The adapter reads only redacted metadata from env-provided local suite artifacts or repo-local public fixtures and writes:

```text
public/data/public-suite-feed.json
```

Forbidden content classes stay out of the feed: finance rows, meeting videos, meeting transcripts, screenshots, X raw text, credentials, secrets, private docs, private costs, and personal raw records.

## Local

```bash
npm install
npm run lint
npm test
npm run build
npm run dev
```
